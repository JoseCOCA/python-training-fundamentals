# S02 — Código de la sesión

Demos ejecutables de los conceptos de S02.

```bash
uv run python main.py
```

`main.py` recorre en orden: condicionales, truthiness, las tres formas de `for`, `break`/`continue`, `for/else`, y comprensiones.

Después de ejecutarlo, modifícalo: cambia los rangos, prueba qué pasa si quitas el `break`, transforma alguna comprensión a `for` y al revés.
