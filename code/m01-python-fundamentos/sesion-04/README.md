# S04 — Código de la sesión

```bash
uv run python main.py
```

`main.py` recorre: función básica, múltiples retornos vía tupla, defaults y argumentos por nombre, el gotcha de `default=[]` con su versión correcta, `*args`/`**kwargs`, scope LEGB, y la diferencia entre función pura e impura.

Después de ejecutarlo, modifica las funciones — agrega validaciones, hazlas más estrictas, prueba qué pasa si pasas tipos inesperados.
