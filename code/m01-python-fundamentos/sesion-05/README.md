# S05 — Código de la sesión

```bash
uv run python main.py
```

`main.py` recorre indexing/slicing, f-strings, métodos esenciales (`strip`, `lower`, `replace`, `split`, `join`, `startswith`, `find`), parseo de líneas tipo catálogo, comparación `join` vs `+=`, y encoding UTF-8.

Después de ejecutarlo, modifica los format specifiers, prueba métodos que no estén en la demo (`title()`, `capitalize()`, `count()`), y rompe parseos con líneas mal formadas para ver qué errores salen.
